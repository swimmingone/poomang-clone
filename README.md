### 처음 프로젝트 환경 셋업하기

```Bash
git clone https://github.com/swimmingone/poomang-todo.git
cd poomang-todo
yarn
yarn husky install
yarn dev
```

- EsLint, Prettier 설정 (tailwind prettier 플러그인 사용)
- husky, lint-staged 로 pre-commit lint checking hook 설정
- 도메인별 디렉토리 구조, atomic design 활용
